# Pixel Clicker

**Pixel Clicker** is a browser-based idle clicker game where players earn points by clicking and purchasing upgrades.  
The game features a pixel-style UI and multiple visual themes.

---

## 🎮 Gameplay

- Click the **Click** button to earn points.
- Purchase upgrades to increase your **score multiplier**.
- The upgrade cost increases each time you buy one.
- Your progress is automatically **saved using localStorage**.

---

## 🌈 Themes

Pixel Clicker includes **4 visual themes**:

- **Light Theme** – A bright white and grey pixel interface.
- **Dark Theme** – A dark black and grey style.
- **Nature Theme** – Green natural colors with bright tones.
- **Yellowish Theme** – A yellow-green palette similar to nature colors.

You can switch themes using the **theme button** in the interface.

---

## 💾 Saving System

The game uses the browser's **localStorage** to store:

- Score
- Upgrade multiplier
- Upgrade price
- Selected theme

This allows progress to remain even after refreshing the page.
